# Installation

## Vagrant
To run the game in a VM, you need to install vagrant, vagrant-hostsupdater and virtualbox. Then run:
```
$ vagrant up
$ make run
```
Then navigate to http://jumpnrun.vm in your browser to play.

# Test
Run `grunt test` to test.
