# Installation

## Vagrant
To run the game in a VM, you need to install vagrant and virtualbox. Then run:
```
$ make vagrant
$ make run
```
Then navigate to http://jumpnrun.vm in your browser to play.

## Requirements
* nodejs
* broccoli-cli
* bower

install with `npm install -g [package]`
